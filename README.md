# ⭐ Star Style Barbershop - Digital Gift Program

یک پلتفرم دیجیتال برای برنامه هدایای خوش‌آمدگویی آرایشگاه استار استایل.

## 🎯 قابلیت‌های اصلی

- 📱 **ثبت‌نام مشتریان** - ثبت‌نام آسان با انتخاب هدیه
- 🎁 **سه گزینه هدیه**:
  - سشوار و استایل مو (رایگان)
  - فید ریش / سایه (رایگان)
  - ۹۹ هزار تومن اعتبار برای تمامی خدمات
- 📊 **پنل مدیریت** - کنترل مشتریان و وضعیت جوایز
- 🌓 **تم Light و Dark** - تجربه کاربری بهتر
- 📱 **Responsive Design** - تمام دستگاه‌ها پشتیبانی می‌شود

## 🛠️ تکنولوژی‌های استفاده‌شده

- **React 19** - UI Framework
- **TypeScript** - Type Safety
- **Tailwind CSS** - Styling
- **Vite** - Build Tool
- **Motion (Framer Motion)** - Animations
- **Express.js** - Backend Server
- **Iran Sans Font** - فونت فارسی حرفه‌ای

## 📦 نصب و راه‌اندازی

### پیش‌نیازها
- Node.js 18+
- npm یا yarn

### مراحل
```bash
# کلون کردن repository
git clone https://github.com/mamati1379/star-style-barbershop.git
cd star-style-barbershop

# نصب dependencies
npm install

# اجرای در حالت development
npm run dev

# Build برای production
npm run build

# شروع server
npm start
```
